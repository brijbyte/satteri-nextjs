# Milestone 2 — withSatteri config wrapper

**This whole page is a real `.mdx` file** (`app/milestone-2/page.mdx`), compiled
by the satteri loader because `next.config.mjs` is wrapped with `withSatteri(...)`.
No `evaluate` shim, no per-page wiring — Next routed and rendered it directly.

## What the wrapper did

- Extended `pageExtensions` so `.md`/`.mdx` count as pages.
- Registered the loader for both **Turbopack** and **webpack**.
- Compiled this file's Markdown to a React module at build time.
